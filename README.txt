README
FileLoad version 1.0
Тестовый проект, написанный в процессе разбора устройства MVC.

Установка:
1. Выполнить скрипт "data/instalScript.sql"
2. Настроить конфигурацю БД "configDB.php"

----------------------------------------------------------------------------

1. Main - главный контроллер, содержащий actions = login, register, logout.
   Load - контроллер загрузки файлов.

2. Для неавторизированных пользователей отображается страница авторизации.
    Регистация станет доступа по нажатию "Sign Up Here" в версии 1.1.
	Страница регистрации(авторизации) содержит два поля:
	   - Email(Адрес электронной почты должен содержать символ "@", обязателен для заполнения)
	   - Pass(не менее 5 символов, обязателен для заполнения).
	   Тестовые данные: Email="admin@admin";  Pass="admin".
	При регистрации(сохранении в БД) и авторизации пароль хешируется MD5.
3. При авторизации(или регистрации) доступна страница работы с файлами(В меню - "Load File").
	Она содержит форму загрузки файла и список загруженных файлов пользователя.
	Таблица с файлами содержит поля - Id file, path, Link for download и date load(дата загрузки-информативно, для проверки правильности сортировки)
	Файлы сортируются по дате загрузки - LIFO.
	При нажатии на "download" файл скачается.
4. Максимальный размер загружаемых файлов - 100 КБ.
	В случае удачной загрузки появится сообщение "File successfully downloaded", иначе разные сообщения об ошибках, например:
		- "File very big! Max size is 100KB!";
		- "File has been loaded!"(Файл уже загружен);
		- "Select a File"( Выбирите файл, если файл не выбран).
	Загрузка файла ощуществляется через "load/loadFile".
5.В правом углу доступна кнопка "Log out" для выхода из системы для авторизированных пользователей и "Log in" для неавторизированных.
